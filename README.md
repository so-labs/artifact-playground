# Artifact Playground

ブラウザでもアプリでも動く、ちょっとした実験ツールを集めています。

## 🛠 収録ツール

- **20% Off**:
  - 入力したテキストから、指定した割合（％）の文字をランダムに削り落とすツール。
- **ウエイトオーバー**:
  - 文字数上限を意識しながら書くためのリアルタイム文字数カウンター。上限に近づくと画面が警告してくれます。
- **スライスドロップ**:
  - 長い文章を指定した上限文字数で自動的に分割し、ページごとに切り替えて個別にコピーできるツール。長文をSNS等に小分けにして投稿する際に便利です。
- **ノリノリ音符**:
  - 文章の改行を整理して、各フレーズの末尾にランダムな音符（♪、♫、♬）をくっつけるツール。文章を強制的に陽気な雰囲気にします。
- **アウトライン・スタジオ**:
  - Markdown の見出し構造を操作・抽出するワークベンチ。アウトライン表示、見出しレベルの調整・移動、セクション単位のスマートコピー（全文・エリア・セクション、引用・コード形式）ができます。PWAとしてインストールすると、OSによっては共有メニューなどから直接ファイルを開けます。

## 💡 特徴

- Vanilla HTML / CSS / JS で構築
- 各ツールはコンポーネント化され、選択時に動的に読み込まれます
- ダークモード（システム設定連動）対応
- PWA対応（ホーム画面に追加してアプリとして使えます）

## 🧪 テストの実行方法

各ツールのコアロジックに対するユニットテストが用意されています。

- **ブラウザで実行**:
  - `tests/index.html` をブラウザで直接開くか、サイドバー左下のフラスコ型アイコン 🧪 をクリックすると、テストランナーが起動してテスト結果が表示されます。
  - 特殊な開発ツールや環境構築（Node.js、npm など）を必要とせず、ブラウザ単体で手軽に動作確認が可能です。
