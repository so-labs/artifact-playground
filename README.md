# Artifact Playground

ブラウザ上でサクッと動く、ちょっとした実験ツールを集めています。

## 🛠 収録ツール

- **20% Off**:
  - 入力したテキストから、指定した割合（％）の文字をランダムに削り落とすツール。
- **ウエイトオーバー**:
  - 文字数上限を意識しながら書くためのリアルタイム文字数カウンター。上限に近づくと画面が警告してくれます。
- **ノリノリ音符**:
  - 文章の改行を整理して、各フレーズの末尾にランダムな音符（♪、♫、♬）をくっつけるツール。文章を強制的に陽気な雰囲気にします。

## 💡 特徴

- Vanilla HTML / CSS / JS で構築
- 各ツールはコンポーネント化され、選択時に動的に読み込まれます
- ダークモード（システム設定連動）対応
